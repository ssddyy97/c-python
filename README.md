

概要

本プロジェクトは、C言語とPythonを利用したプログラミング学習プロジェクトです。
基本的なアルゴリズムやデータ処理を実装し、低レベル言語であるC言語と高水準言語であるPythonの特徴や違いを理解することを目的としています。

また、両言語の処理速度や実装方法の違いを比較しながらプログラミングスキルを向上させることを目標としています。
